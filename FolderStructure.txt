We present a web-based platform that enables secure, domain-specific API key
generation powered by Retrieval-Augmented Generation (RAG) and opensource large language models (LLMs) such as LLaMA 3, Mixtral, and Gemma.
• Users can upload custom data or documents, which are embedded and stored in
a vector database. When users interact with the system, a chatbot interface
leverages semantic search to retrieve relevant content and answer queries. Based
on the request, the system dynamically selects an appropriate LLM and issues an
API key with defined scopes and permissions.
AIM:
To build a web application that generates domain-specific API keys using
RAG , open source LLMs
OBJECTIVES:

AIM & OBJECTIVES OF THE PROJECT
(Problem Statement)
• To implement a Retrieval-Augmented Generation (RAG) pipeline for
answering queries from uploaded documents.
• To integrate open-source LLMs like LLaMA 3, Mixtral, and Gemma for
flexible model selection.
• To enable domain-specific API key generation with customizable scopes
and permissions.
• To provide a secure, user-friendly web interface for document upload
and chatbot interaction.
• To manage and restrict API access based on domain, usage, and
expiration policies.
